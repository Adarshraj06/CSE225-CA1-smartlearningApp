# CSE225-CA1-smartlearningApp

A simple Android application built using **Kotlin** and **Jetpack Compose** that demonstrates splash screen implementation, navigation between activities, and data passing.

---

## 🚀 Features

* ⏳ Splash Screen with loading indicator
* 🔄 Automatic navigation using `LaunchedEffect`
* 📤 Data passing between activities using Intent
* 🏠 Dashboard screen with user welcome message
* 🔙 Back navigation to previous screen
* 🎨 Built using modern UI toolkit (Jetpack Compose)

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **UI:** Jetpack Compose
* **IDE:** Android Studio
* **Concepts Used:**

  * Activity Navigation
  * Intent (Data Passing)
  * State Management
  * Compose UI

---

## 📂 Project Structure

* `MainActivity` → Splash Screen
* `DashboardActivity` → Displays user name
* `SecondActivity` → Demonstrates data receiving and back navigation

## ⚙️ How It Works

1. App starts with a **Splash Screen**
2. After a delay, it navigates to **Dashboard**
3. Username is passed using **Intent**
4. User can navigate to next screen and return back
